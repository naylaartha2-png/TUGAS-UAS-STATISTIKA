Tugas ini bertujuan untuk menerapkan metode Regresi Linear Berganda dalam menganalisis faktor-faktor yang memengaruhi tingkat risiko penduduk terkena tindak pidana di Indonesia.Melalui data yang mencakup jumlah kejahatan, persentase penyelesaian perkara, dan selang waktu terjadinya kejahatan menurut provinsi tahun 2017–2021, dilakukan proses pengumpulan data, eksplorasi data, pembersihan data (preprocessing), serta pembangunan model regresi untuk memprediksi risiko kejahatan per 100.000 penduduk.Hasil analisis diharapkan dapat memberikan informasi mengenai pengaruh masing-masing variabel terhadap risiko kejahatan sehingga dapat menjadi dasar pengambilan keputusan dalam meningkatkan keamanan dan efektivitas penanganan tindak pidana di setiap provinsi di Indonesia.
